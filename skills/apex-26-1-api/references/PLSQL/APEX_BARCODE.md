# APEX_BARCODE

Source: [Oracle APEX 26.1 API Reference](https://docs.oracle.com/en/database/oracle/apex/26.1/aeapi/APEX_BARCODE.html)

Status: curated first-pass reference.

## Purpose

`APEX_BARCODE` generates barcode and QR code images in SVG or PNG format. It supports QR Code, Code 128, and EAN-8 variants.

Use it to render scannable identifiers, URLs, tickets, labels, inventory tags, and workflow handoff codes in APEX pages or generated documents.

## API Surface

| Need | Members |
| --- | --- |
| QR Code | `GET_QRCODE_SVG`, `GET_QRCODE_PNG` |
| Code 128 | `GET_CODE128_SVG`, `GET_CODE128_PNG` |
| EAN-8 | `GET_EAN8_SVG`, `GET_EAN8_PNG` |

## Render QR Code SVG Inline

```sql
declare
    l_svg clob;
begin
    l_svg := apex_barcode.get_qrcode_svg(
        p_value            => apex_page.get_url(
                                  p_page   => 20,
                                  p_items  => 'P20_ORDER_ID',
                                  p_values => '1001'),
        p_size             => 220,
        p_quiet            => 2,
        p_foreground_color => '#000000',
        p_background_color => '#FFFFFF');

    sys.htp.p('<div class="qr-code">');
    sys.htp.p(l_svg);
    sys.htp.p('</div>');
end;
/
```

The SVG is generated by APEX, but the encoded value can contain user data. Validate or construct values deliberately.

## Download QR Code PNG

```sql
declare
    l_png blob;
begin
    l_png := apex_barcode.get_qrcode_png(
        p_value => 'ORDER-1001',
        p_size  => 256);

    apex_http.download(
        p_blob         => l_png,
        p_content_type => 'image/png',
        p_filename     => 'order-1001-qr.png');

    apex_application.stop_apex_engine;
end;
/
```

## Code 128 Label

```sql
declare
    l_svg clob;
begin
    l_svg := apex_barcode.get_code128_svg(
        p_value => 'SHIP-2026-0001');

    sys.htp.p(l_svg);
end;
/
```

Use Code 128 for general alphanumeric identifiers. Use EAN-8 only for valid EAN-8 values.

## Safety Notes

- Do not encode secrets in QR codes or barcodes.
- Use APEX-generated URLs for application links instead of concatenating `f?p` strings by hand.
- Test scan reliability at the printed/displayed size.
- For QR codes, use quiet zones and sufficient error correction for printed labels.
- Download PNG output through `APEX_HTTP.DOWNLOAD` and stop the APEX engine afterward.
