## Purpose

This procedure enables you to write a procedure to submit feedback, rather than using the feedback page generated by Create Page Wizard.

Use this page when code needs the `APEX_UTIL.SUBMIT_FEEDBACK` procedure. Confirm security, workspace, and session requirements for your calling context.
